# Ames Housing Price Analysis – MATH 237 Project 1

### Overview
This project investigates the main factors that influence house prices in Ames, Iowa.  
It explores how features such as neighborhood, overall quality, and year built affect the sale price of a house.  
The goal is to understand which characteristics have the most significant impact on housing prices and to build a simple regression model to predict them.

This analysis is useful for homebuyers, sellers, and builders who want insights into pricing trends and property value drivers in the Ames area.

---

### Dataset Description
The Ames Housing Dataset contains information describing the sale of housing properties in Ames, Iowa, from 2006 to 2010.  
It consists of 1,460 observations and 80 features representing various home characteristics, such as structural quality, living area, and neighborhood location.

- Source: [Kaggle – House Prices: Advanced Regression Techniques](https://www.kaggle.com/c/house-prices-advanced-regression-techniques/data)  
- Original Reference: [DeCock, D. (2011). *Ames, Iowa: Alternative to the Boston Housing Data.* *Journal of Statistics Education* 19(3)](https://jse.amstat.org/v19n3/decock.pdf)

---

### Objectives
1. Identify the most important features that influence house prices.  
2. Determine whether newer houses sell for more than older ones, controlling for other variables.  
3. Compare neighborhoods to see which have the highest average home prices and what features contribute to that difference.

---

### Ethical Note
This dataset represents housing data only from Ames, Iowa, and results may not generalize to other regions.  
Models built on this data may also reflect existing socioeconomic and geographic biases.
